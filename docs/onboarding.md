# Onboarding
Deze workshop gaat uit van dat je het volgende hebt geinstalleerd op je laptop

1. JDK 17
2. IDE waarbinnen je Java kan programmeren


## Optioneel (voor werkende C4 diagrammen, zie ModulithTest.java)

1. Installeer `sudo apt install graphviz`
2. Plaats de Graphviz bin folder op het pad van de PlantUML plugin in je IDE
   * In IntelliJ: zoek op plantuml -> plantuml onder languages en frameworks -> vul bij "Additional..." tekstbox in `/usr/bin/dot`
